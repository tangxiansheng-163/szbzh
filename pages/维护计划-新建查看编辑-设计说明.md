## 一、页面关系与交互

```
equipment-maintenance-plan.html（维护计划列表）
    │
    ├── [+ 新建计划] ──→ plan-create-modal 弹窗 ──→ equipment-maintenance-plan-create.html
    │
    ├── [查看] ──→ list-view-modal 弹窗 ──→ equipment-maintenance-plan-detail.html?id=xxx
    │
    └── [编辑] ──→ edit-modal 弹窗 ──→ equipment-maintenance-plan-edit.html?id=xxx
```

- **新建计划**：点击列表页「+ 新建计划」按钮，在 `plan-create-modal` 弹窗中加载新建表单
- **查看**：点击列表行「查看」按钮，在 `list-view-modal` 弹窗中加载详情页（只读）
- **编辑**：点击列表行「编辑」按钮，在 `edit-modal` 弹窗中加载编辑表单

## 二、新建维护计划（equipment-maintenance-plan-create.html）

### 2.1 页面定位

创建设备维护计划，支持巡检计划、保养计划、预防性维修计划；计划编号自动生成。

### 2.2 基本信息字段

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 计划名称 | 文本 | 是 | 如：空压机A定期保养 |
| 计划类型 | 下拉 | 是 | 巡检计划 / 保养计划 / 维修计划 |
| 设备编码 | 文本 | 是 | 关联设备 |
| 设备名称 | 文本 | 是 | 设备名称 |
| 执行周期 | 下拉 | 是 | 每日 / 每周 / 每月 / 每季度 |
| 责任人 | 文本 | 是 | 计划责任人 |
| 计划开始日期 | 日期 | 是 | 首次执行日期 |
| 计划结束日期 | 日期 | 否 | 不填表示长期有效 |
| 维护内容说明 | 多行 | 否 | 维护/巡检内容、注意事项等 |
| 备注 | 多行 | 否 | 其他补充说明 |

### 2.3 维护项目明细

| 列 | 说明 |
|----|------|
| 序号 | 维护项目序号 |
| 维护项目 | 如：润滑油检查、滤芯更换 |
| 检查标准/要求 | 检查标准、合格判定 |

支持「+ 添加维护项目」、行内「删除」。

### 2.4 操作按钮

- **取消**：关闭弹窗，不保存
- **保存草稿**：保存为草稿状态，关闭弹窗并返回列表
- **保存并提交审批**：校验必填项后提交审批，关闭弹窗并返回列表

## 三、查看维护计划详情（equipment-maintenance-plan-detail.html）

### 3.1 页面定位

只读展示维护计划完整信息及执行记录。

### 3.2 基本信息卡片

| 字段 | 说明 |
|------|------|
| 计划编号、计划名称、计划类型、计划状态 | 计划核心信息 |
| 设备编码、设备名称、执行周期、责任人 | 关联设备与执行信息 |
| 下次执行、创建人、创建时间 | 执行与审计信息 |
| 维护内容说明 | 长文本展示 |

### 3.3 维护项目明细卡片

表格展示：序号、维护项目、检查标准/要求。

### 3.4 执行记录卡片

最近 5 条执行记录：执行日期、执行人、执行结果（已完成/未完成等）、备注。

### 3.5 数据来源

通过 URL 参数 `id` 或 `window.__modalContentId` 获取计划 ID，使用 Mock 数据按 ID 渲染。实际实现时对接后端接口。

## 四、编辑维护计划（equipment-maintenance-plan-edit.html）

### 4.1 页面定位

编辑已有维护计划，仅草稿、已暂停、已停用等可编辑；已启用计划编辑规则以业务为准。

### 4.2 表单字段

与新建表单结构一致，以下为只读：计划编号、计划状态。其余字段可编辑。

### 4.3 维护项目明细

支持新增、删除、修改明细行，与新建表单一致。

### 4.4 操作按钮

- **取消**：关闭弹窗，不保存
- **保存**：校验后保存，关闭弹窗

## 五、状态与计划类型枚举

### 5.1 计划状态

| 值 | 展示 |
|----|------|
| draft | 草稿 |
| pending | 待审批 |
| approved | 已审批 |
| enabled | 已启用 |
| paused | 已暂停 |
| disabled | 已停用 |

### 5.2 计划类型

| 值 | 展示 |
|----|------|
| inspection | 巡检计划 |
| maintenance | 保养计划 |
| repair | 维修计划 |

### 5.3 执行周期

| 值 | 展示 |
|----|------|
| daily | 每日 |
| weekly | 每周 |
| monthly | 每月 |
| quarterly | 每季度 |

## 六、技术实现说明

### 6.1 新增/修改文件

| 文件 | 说明 |
