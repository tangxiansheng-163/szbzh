# 本地服务说明

## 端口与入口

| 配置项 | 说明 |
|--------|------|
| **默认端口** | `8080` |
| **系统入口** | `http://localhost:8080/main.html` |
| **欢迎页** | `http://localhost:8080/index.html`（会跳转到 main） |

服务根目录为 **原型文件夹**（即 `main.html`、`pages/`、`assets/` 所在目录），避免访问 `main.html` 出现 404。

---

## 启动方式

### 方式一：批处理（推荐）

在**资源管理器**中进入 **原型** 目录，双击 **`start-server.bat`**：

- 自动检测 Python 是否可用
- 在**当前项目目录**启动服务（端口 8080）
- 启动后**勿关闭**弹出的命令行窗口

### 方式二：VBS 脚本（一键启动 + 打开浏览器）

双击 **`启动服务器.vbs`**：

- 启动 `start_server.py` 并打开命令行窗口
- 约 2.5 秒后自动在浏览器打开 `http://localhost:8080/main.html`

### 方式三：命令行

在 **命令提示符** 或 **PowerShell** 中，进入原型目录后执行：

```bash
cd /d "d:\...\制造云-标准化\设计协同\md文件\原型"
python start_server.py
```

或使用环境变量指定端口（如 8888）：

```bash
set SERVER_PORT=8888
python start_server.py
```

然后访问 `http://localhost:8888/main.html`。

---

## 依赖与要求

- **Python 3**：已安装并加入系统 `PATH`
- **端口 8080**：未被其他程序占用（否则修改 `SERVER_PORT` 或 `start_server.py` 中的 `PORT`）

---

## 常见问题

1. **404 文件未找到**  
   确保使用 **`start_server.py`** 或 **`start-server.bat`** 启动，且未移动过原型文件夹。勿在上级目录运行 `python -m http.server`。

2. **端口已被占用**  
   设置 `SERVER_PORT` 为其他端口（如 8888）后重新启动，或关闭占用 8080 的程序。

3. **未找到 Python**  
   安装 Python 3 并勾选 “Add to PATH”，或手动将 `python.exe` 所在目录加入系统环境变量。

4. **双击 .bat 闪退**  
   在资源管理器地址栏输入 `cmd` 回车，在该目录下手动执行 `start-server.bat`，查看命令行中的报错信息。

5. **在 IDE 终端启动报错（中文路径等）**  
   请直接在**资源管理器**中打开原型目录，双击 `start-server.bat` 或 `启动服务器.vbs` 启动。
